## Licencia

Este proyecto utiliza la **Licencia MIT**, seleccionada al crear el repositorio. Esta licencia permite a cualquier persona usar, copiar, modificar, fusionar, publicar, distribuir, sublicenciar o incluso vender el software

## ¿Qué permite la licencia MIT?

- Usar el software libremente  
- Copiarlo y modificarlo  
- Distribuirlo  
- Usarlo de manera comercial  
- Integrarlo en software privado

## ¿Qué obligaciones tengo?

- Debes **mantener la nota de copyright** y la **nota de licencia** en cualquier copia del software.

## ¿Qué no ofrece?
